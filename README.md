# iOS 元書輸入法 - 蝦米方案

基於元書輸入法的蝦米輸入方案，提供多種配置選項和精美皮膚主題，讓您享受流暢的蝦米輸入體驗。

## 主要特色

- 🚀 **完整蝦米功能**：包含標準字碼、簡碼及 VRSF 選字等完整功能
- 🎨 **皮膚主題**：提供實用皮膚，支援淺色/深色主題
- 📝 **智慧造詞**：支援臨時造詞（註：「屏蔽無效鍵」方案無此功能）和自訂詞庫編輯
- 🔍 **查詢模式**：讀音查詢、同音字查詢
- 🎯 **快打模式**：提示可用簡碼，幫助學習和提升輸入效率
- 🔤 **豐富符號**：50+ 分類符號清單、變體英數、字母變化形
- 📅 **日期時間**：多格式日期時間輸出
- 🔠 **拼音輸入**：漢語拼音輸入，支援數字聲調
- 🌐 **英文詞庫**：完整版包含英文詞庫，支援自動補全和大小寫轉換
- 🛡️ **無效鍵屏蔽**：可選擇屏蔽無效按鍵，避免誤觸

## 方案配置

本方案提供多種配置選擇，滿足不同使用需求：

### 基本配置

- **中文輸入**：純中文輸入，不含英文詞庫
- **中文輸入+英文詞庫**：包含英文詞庫，支援英文自動補全
- **中文輸入+屏蔽無效鍵**：純中文輸入 + 屏蔽無效按鍵
- **中文輸入+英文詞庫+屏蔽無效鍵**：完整功能 + 屏蔽無效按鍵

### 皮膚主題

- **蝦米輸入法皮膚設計器**：https://ryanwuson.github.io/rime-liur-ios-skin/
- 使用者可以使用習慣調整：

  - **按鍵佈局**：包含「空白鍵長度」、「左、右手持機」、「上下滑功能」及「長按功能」
  - **工具配置**：包含25項實用工具，使用者可依需求於工具列放置10項
  - **基礎外觀**：調整亮/暗色鍵盤顏色
  - **進階微調**：細緻調整「26鍵」、「數字」、「符號」及「Emoji」鍵盤。
- 編輯皮膚後，請依序：

  - 按右上角 **i 圖示**，編輯皮膚名稱及作者資訊

  - 按右上角 **導出配置 (.cskin)**，即可導出自定皮膚，包含亮暗色26鍵 **自動截圖**

## 安裝方式

### 系統需求

- **iOS/iPadOS**: 元書輸入法 App

- 支援 iOS 16 或以上版本

- 下載位置：

  https://apps.apple.com/us/app/%E5%85%83%E4%B9%A6%E8%BE%93%E5%85%A5%E6%B3%95/id6744464701?l=zh-Hant-TW

### 安裝步驟

1. **選擇適合方案並複製網址**
   
   - **中文輸入**：純中文輸入，不含英文詞庫

     https://codeload.github.com/ryanwuson/rime-liur-ios/zip/refs/heads/main
   
   - **中文輸入+英文詞庫**：包含英文詞庫，支援英文自動補全
   
     https://codeload.github.com/ryanwuson/rime-liur-ios/zip/refs/heads/english
   
   - **中文輸入+屏蔽無效鍵**：純中文輸入 + 屏蔽無效按鍵
   
     https://codeload.github.com/ryanwuson/rime-liur-ios/zip/refs/heads/main-block
   
   - **中文輸入+英文詞庫+屏蔽無效鍵**：完整功能 + 屏蔽無效按鍵
   
     https://codeload.github.com/ryanwuson/rime-liur-ios/zip/refs/heads/english-block
   
2. **匯入元書輸入法**
   - 開啟元書輸入法 App
   - 選擇「輸入方案」
   - 點擊「下載方案」
   - 點擊「㊉」
   - 輸入「方案名稱(自訂)」
   - 貼上「下載網址」
   - 點擊右方綠色按鍵下載，等待下載完成
   - 選擇「輸入方案」
   - 點擊「方案目錄切換」
   - 依序選擇「RimeUserData」
   - 選擇「方案名稱(自訂)」
   - 選擇「rime-liur-ios」
   - 點擊「打開」
   
3. **安裝皮膚主題**
   
   - **從皮膚設計器中直接匯出：**
   
     - 打開 iOS 瀏覽器（以 Safari 為例），訪問**蝦米輸入法皮膚設計器**
   
       https://ryanwuson.github.io/rime-liur-ios-skin/
   
     - 按右上角 **導出配置 (.cskin)**，即會出現下載視窗
   
     - 將皮膚儲存至「檔案.app」
   
   - 點擊.cskin檔案，以元書輸入法 App 開啟
   
   - 長按「皮膚主題」，點擊 **「運行 main.jsonnet」**，即可使用

## 基本使用

### 蝦米字碼輸入

- 直接輸入蝦米字碼，按 `Space` 出字，或下滑 Q ~ P 鍵，可以 1 ~ 9 數字鍵選字
- 支援標準字碼、簡碼輸入
- `lime.db` 的 `custom_user` 只匯入多字詞到加詞檔，且數字字根會轉成 `0123456789 -> versfwlcbk`
- 備份資料庫匯入加詞時，可用 `scripts/import_backup_db_missing_words.py` 只匯入主字典沒有的多字詞，並自動整理折行、空白與特殊字
- 方案名稱目前顯示為 `嘸蝦米`，方便在小狼毫方案選單中辨識
- 小狼毫候選過長時改用內建縮寫顯示，選字仍會上完整內容
- 小狼毫目前改成鍵盤優先選字：滑鼠懸停不改變候選，滾輪只翻頁
- 候選浮動窗開啟時，數字小鍵盤 `1~9` 會直接選字，不再誤輸出數字
- 主翻譯器已開啟選字學習；`related` 也會在你明確選第 2、3、4 候選時加重學習
- `related` 學習已調成更明顯模式，常選候選會更快往前
- 可直接輸入 `,,字根`，按 Enter 後輸入詞句，完成後再按 Enter 寫入加詞檔
- 具備 VRSF 選字功能

### 英文詞庫輸入（完整版限定）

- **英文詞庫版本**：英文單詞會自動出現在候選選單，按 `Space` 出字
- **大小寫轉換**：
  - 輸入英文後按 `]` → 首字母大寫
  - 輸入英文後按 `]]` → 全部大寫

### 特殊功能

| 功能 | 快捷鍵/引導鍵 | 說明 | 備註 |
|:-----|:-------------|:-----|:-----|
| 造詞模式 | `;` | 臨時造詞功能 | 「屏蔽無效鍵」方案不適用 |
| 同音選字 | 左下角`「同音」`鍵或L上滑`'` | 打完字碼後直接按左下角`「同音」`或上滑L帶出 `'` | 需先鍵入蝦米字碼 |
| 讀音查詢 | `;;` | 輸入字碼查讀音 | 需開啟元書輸入法「候選Comment功能」 |
| 拼音輸入 | `;'` | 漢語拼音輸入中文 | 下滑QWERT可對應1～5聲調 |
| 快打模式 | `,,sp` | 提示可用簡碼 | 需開啟元書輸入法「候選Comment功能」 |
| 按鍵說明 | `,,h` | 顯示所有快捷鍵說明 |  |

## 無效鍵屏蔽

帶有「屏蔽無效鍵」的配置提供以下功能：

- **智慧屏蔽**：根據當前輸入狀態屏蔽無效按鍵
- **避免誤觸**：減少輸入錯誤，提升輸入準確度
